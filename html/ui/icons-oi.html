Using Open Iconic

Recommended for inline use

Add a class oi oi-icon-name to add a Open Iconic Font icon anywhere.

For example <i class="oi oi-home"> generates

Inside Buttons

Add a class oi oi-icon-name to i or span to add a Open Iconic icon inside a button.
